## 1) Техническое задание (ТЗ)

Цель: реализовать модуль распознавания жестов руки на PyTorch, который получает `21` landmark от MediaPipe и возвращает стабильный класс жеста для управления AirPainter.

### 1.1 Классы жестов (v1)
- `0`: `pause`
- `1`: `draw`
- `2`: `erase`
- `3`: `clear`
- `4`: `change_color`

### 1.2 Вход и выход модели
- Вход: landmarks в формате `[x1, y1, x2, y2, ... x21, y21]`.
- Предобработка:
  - сдвиг относительно запястья (`landmark 0`),
  - нормализация на размер ладони,
  - дополнительные признаки: ключевые расстояния между пальцами.
- Выход: вероятности по 5 классам + `argmax` класс.

### 1.3 Контракт интеграции с Part 1
- Part 1 передает landmarks каждый кадр.
- Part 2 возвращает:
  - `gesture_id`,
  - `gesture_name`,
  - `confidence`,
  - `stable_gesture` (после сглаживания).

### 1.4 Критерии готовности v1
- Accuracy на валидации: целевой порог `>= 90%`.
- Инференс на CPU: целевой порог `<= 10 ms` на кадр.
- Стабилизация предсказаний:
  - окно `N` кадров,
  - подавление одиночных шумовых скачков,
  - подтверждение жеста по большинству/средней вероятности.
